1. Start program
2. Display welcome message and main menu
3. User selects an option:
   a. Add new fault record(s)
      - Input fault details
      - Validate voltage/current
      - Classify severity
      - Store record
   b. View all recorded faults
      - Display each record
      - Show summary
   c. Generate & save report
      - Write all records to electrical_fault_report.txt
      - Write summary statistics
   d. Exit program
4. Repeat menu until user chooses Exit
5. End program
